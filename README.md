Personal Portfolio Website

A simple and responsive personal portfolio website created using HTML5. This project showcases personal information, technical skills, and contact details using semantic HTML elements and internal navigation links.

📌 Features
HTML5 semantic structure
About section
Skills section
Contact form with validation
Internal navigation links
Profile image support
Beginner-friendly project structure
🛠️ Technologies Used
HTML5
VS Code
Web Browser (Chrome/Firefox)

🚀 Setup Instructions
Step 1: Clone or Download the Project

Download the project files or clone the repository.

git clone <repository-link>
Step 2: Open in VS Code

Open the project folder in Visual Studio Code.

Step 3: Run the Project

Open the index.html file in any browser.

OR

Use the Live Server extension in VS Code:

Right click on index.html
Select Open with Live Server
